An offline, desktop-based learning tool built with Python and CustomTkinter to help Computer Science students practice SQL syntax in a safe, hands-on environment.

## :rocket: The Vision

As a facilitator of High School Computer Science, I realized that many students are forced to memorize SQL syntax, be it for practical or theory exams--often without any real oppotuinity to use it from scratch. They'd memorize, take the test and move. This project arims to bridge that gap by providing a plug-and-play sandbox where SQL finally makes sense through hands-on practice.

## :stars: Features

- **Zero-Setup SQL:** Uses SQLite, so no database server (like MySQL) installation is required.
- **Instant Feedback:** Type a query and see the results (or helpful error messages) immediately
- **Offline First:** Designed for environments with limited internet connectivity.
- **Modern UI:** Built using CustomTkinter for a sleek, dark-mode friendly experience.

## :tools: Tech Stack

- **Language:** Python 3.x
- **GUI Library:** [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)
- **Database:** SQLite3 (Standard Python Library)
